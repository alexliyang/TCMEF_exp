中原慈善基金

中原慈善基金属香港注册之非牟利慈善组织，于2004年10月13日根据香港公司法例而成立，致力扶助香港及中国地区之贫穷人士及推动教育活动，主要经费来自中原集团。

中原慈善基金基金使命

编辑
中原慈善基金正致力达成以下目标︰[1] 
  1. 为香港及国内的贫困及弱势社群提供援助，协助他们渡过难关，改善生活；
  2. 培育年幼，借着推广、支持及发展教育活动，帮助发掘香港及国内青少年各方面的潜能。

中原慈善基金组织架构

编辑

中原慈善基金管理委员会

由中原集团成员公司委派员工代表组成。每位管理委员的任期为一年[2]  。
管理委员会扮演选举团的职能，负责提名及选出中原慈善基金之董事会成员[2]  。

中原慈善基金董事会

由两种类别的董事组成：第一类别为中原集团员工；第二类别为外界人士，此类人士必须曾任其他慈善机构的董事职务，而该慈善机构须拥有每年港币500万总额的捐款预算[2]  。
董事会由5至9位董事组成。于董事会内属第一类别的董事成员所占的人数，不可多于第二类别的董事成员[2]  。
董事会成员由管理委员会选出，负责管理中原慈善基金的日常事务以及审批拨款申请，任期为三年[2]  。

中原慈善基金基金资助

编辑

中原慈善基金资助宗旨

  • 我们重点拨款予扶贫及推动教育有关的活动和资助计划[3]  。
  • 我们优先考虑支持非政府资助的项目[3]  。
  • 除奖学金可以个人名义申请之外，一般来说，机构应为注册机构或慈善机构，同时能够提供足够的资料证明过往曾成功举办有关的活动才会获得考虑[3]  。
  • 所有计划均需拥有明确的计划目标、受助对象、运作详情、推行手法、预期成效、推行时间表，以及财政预算案[3]  。
  • 所有计划应以高效率和高效益作为基本原则[3]  。
  • 所有计划需属非牟利性质[3]  。
  • 全年接受拨款申请，每项拨款金额通常不超过HKD$200,000[3]  。

中原慈善基金资助范围

中原慈善基金希望透过拨款予扶贫及推动教育有关的项目和活动，以及透过提供奖学金，达成以上目标[1]  。

中原慈善基金申请流程

编辑

中原慈善基金指引

申请机构须以书面提出申请，将有关该项目的的详尽计划书，其中需包括有关计划目标、受助对象、运作详情、推行手法、预期成效、推行时间表，以及财政预算案等资料，寄交：中原慈善基金有限公司[4] 
中原慈善基金全年接受拨款申请，申请将交由董事会审批，需时约二至三个月，申请机构会获书面通知申请结果。

中原慈善基金拨款条件

获拨款机构必须签署承诺书，保证拨款一定用于中原慈善基金核准的特定目标、用途及推行时间表。获拨款机构亦需事先征得中原慈善基金的同意，才可更改核准计划的推行时间表或财政预算[5]  。

中原慈善基金监察机制

编辑
为确保拨款用得其所，中原慈善基金或会安排代表视察受资助计划的进展，受资助机构需定期提交进展报告，并须于计划完成后两个月内提交详细报告[6]  。
