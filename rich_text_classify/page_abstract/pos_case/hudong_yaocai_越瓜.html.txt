名称 : 越瓜 别称 : 菴瓜、生瓜、梢瓜、酥瓜 学名 : Cucumis melo var.conomon 分布 : 原产于热带亚洲、中国大陆，日本与中国大陆栽培最盛 分类 : 葫芦科 品种 : 越瓜是甜瓜的变种，果皮极薄，成熟之后具香气，但缺乏甜味 型态 : 茎叶密生短毛，具卷须，叶互生，心状或掌状，近5角形，叶缘浅裂或深裂，因品种而异，同一株有雄花及两性花，也有同株雌雄异花，花冠黄色，果实筒状或棒状，熟果肉质酥软，具香气。 用途 : 为食用蔬果，常被制成酱菜，是下饭拌粥开胃的古早味菜 花期 : 4～11月生产，6～8月盛产期 管理 : 性喜高温多湿，生育适温20～30度，15度以下导致生育不良 施肥 : 栽培土质以砂质土壤为佳，排水、日照需良好 繁殖 : 播种法繁殖 (一)概述 越瓜(Oriental pickling melon.) 学名：Cucumis melo L.conomon Group. 葫芦科，甜瓜属，甜瓜种中以嫩果生食的变种，一年生蔓性草本植物，别名生瓜、梢瓜、脆瓜、染色体数2n=2x=24。嫩瓜清脆，每100g鲜果含水分95～96g，碳水化合物2.5～3.4g，维生素摄氏度4～16毫克，还含有矿物质及其他维生素等，适生食，也可腌渍或炒食。 主要分布于中国、日本及东南亚。中国栽培较普遍。一般认为，越瓜、菜瓜与甜瓜起源于同一物种。甜瓜由非洲经中东传入印度进一步分化后，经越南传中国，故名越瓜。6～7世纪由中国传入日本。越瓜在中国自古栽培，公元6世纪30年代左右的《齐民要术》中已有记载，其后宋代已有梢瓜之称。 (二)特征特性 越瓜生长势较强。根系发达。茎蔓性、有棱、分枝性强。叶五角心脏形、深绿色，叶面较皱。多雌、雄同株异花，侧蔓，特别是孙蔓第1、2节多生雌花。果实长圆筒形或椭圆形、果面光滑、长20～30厘米，单果重0.4～2.5千克，果皮绿白、墨绿或有深绿斑纹。果肉白或浅绿色，味淡无香气。越瓜喜温暖。种子发芽最低温度为15℃，适温28～30℃；植株生长发育适温为25～30℃，13℃以下受抑制。地温14℃以下根系生长停滞。耐热性、耐湿性、抗病性较强。其生长发育规律近似甜瓜。 (三)种类与品种 越瓜分生食和加工两个类型：生食类型果皮薄，肉质脆嫩多汁，生食或拌凉菜，也可炒食、加工。如广东的白瓜，各地的酥瓜、梢瓜等。加工类型的果皮较厚，果肉致密，生食略带酸味，如老羊瓜。 著名的越瓜品种有在华东栽培较多的白皮梢瓜、花皮梢瓜。 (四)栽培技术 (1)露地栽培技术 1.栽培时间 越瓜为喜温耐热作物，在栽培时间上以其果实发育成熟期在当地的高温干旱季节最为理想。我国露地最适栽培时间均为春播夏收。华南地区，大多为2～3月份播种，5～6月份收获。黄淮海地区、长江流域及华北地区，为4月播种，7月收获。露地直播，播种期可安排在晚霜过后再出苗的时间。 2.整地施肥 越瓜地的选择，应严格避免重茬，不要在3～5年内种过瓜的地块和轮作年限不够的地块上种植。适于土层深厚排水良好的沙壤土。土壤过分粘重，地下水位太高或低洼地，不适合种植。 整地深翻、施肥后，做成平畦或高畦。 3.播种育苗 (1)直播 北方地区常采用穴播直播法。直播栽培较省工。穴播时，挖深3～4厘米的小穴，放入4～5粒干籽或湿籽，复土1～2厘米即成。若播瓜芽，则需在穴中浇些水，待水渗下后再播种，每穴播2～3粒。每公顷用种量1.5千克左右，每公顷保苗30000～37500株(行距1.0米，株距0.3米)。有条件时应进行她膜复盖。 出苗后应加强中耕，以促进根系深扎。一般松土3～4次。进行地膜复盖栽培时，中耕工
