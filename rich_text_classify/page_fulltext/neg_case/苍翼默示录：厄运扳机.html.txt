将A、B、C三键同时按下可以消耗Heat Gauge50点发动，让自己的行动的硬直立即取消。·防御反击 - 在防御时对手硬直的时间同时，按下 A + B键可以消耗Heat Gauge25点使用，把对手弹开。·击溃防御 - 另外消耗25点Heat Gaug可使出的特殊攻击，只能以BARRIER GUARD挡下。Barrier Gauge防御时同时按下A、B两键会消耗此槽[2]，可强化防御。消耗的Barrier Gauge在一段时间后会慢慢回复。如此槽消耗完毕，会进入DANGER状态，自身受到的伤害会化为1.5倍，在一段时间后会慢慢回复。消极行动对战中若是一段时间不作攻击只是消极逃跑的话，会被付上受到攻击1.5倍的状态。和DANGER状态相加会变成3倍。超限驱动（Overdrive）Overdrive有发动次数的限制(最大两次)：同时按下 A + B + C + D 使用时可以发动Break Burst把对手弹飞(金色光为直接发动。被对手攻击中途使用为绿光)。一击必杀（ASTRAL技）决胜局[2]、对手体力不足20%、自身HG全满时可以使用的特殊招式，打中的话会立即把对手击败。 但一般只有拉格纳·布莱德艾奇、雷琪儿·阿尔卡特、ν-13可使用，其他角色则是故事模式全破后才可使用。苍翼默示录：厄运扳机游戏特色编辑苍翼默示录：厄运扳机画面华丽1280*768的分辨[2]率，全3D背景，人物动作清晰流畅，声光特效丰富。被誉为[史上最华丽的2D格斗游戏]。苍翼默示录：厄运扳机角色性格突出角色的性能差异巨大，每个角色都拥有其自成一派的特殊系统。各角色间的外观、性格属性差异大，重复感较低，各角色各具强烈的特色。苍翼默示录：厄运扳机操作感强与同社出品的著名2D格斗游戏系列《圣骑士之战》相比，同样是用复杂的系统换取优良的手感，不过重新分布了键位功能，降低了上手门槛。招式的发生时间普遍延长，游戏节奏和整体速度减慢。例如本作的轻攻击A的发生时间普遍比《圣骑士之战》的轻攻击P延长了一倍。少见将和RPG系统做结合的格斗游戏。故事背景上有着庞大的世界观，各角色故事路径错综复杂且关系密切。苍翼默示录：厄运扳机游戏测评编辑《苍翼默示录：厄运扳机》是日本动漫格斗游戏代表作，出色的战斗系统和华丽的游戏画面，带给玩家更多的是视觉享受，感受极致光影下的战斗乐趣。游戏角色格局的特色，均衡性做的很好，保持了系统的平衡，属性相克，让玩家们随意搭配技能组合，玩出不同的打法技巧。苍翼默示录：厄运扳机配置要求编辑OS: Microsoft Windows XP Embedded SP2/SP3CPU: Intel LGA 775 CPU.
